# A800X frame fixture

`a800x_v0.4.0.frames` contains the deterministic PreEQ and DRC frames emitted
by the verified BP10 DSP Controller 0.4.0 source state. The private development
tree had already compared this state byte-for-byte with the established 0.3.7
A800X baseline. The public test uses this frozen fixture so it does not depend
on private Git history.
